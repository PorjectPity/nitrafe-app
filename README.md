## ✨ Caracteristici Principale / Key Features

### 🧹 1. Clean & System Residual Purge
- **Purjare Temp & Cache**: Curăță fișierele temporare Windows (`%TEMP%`, `C:\Windows\Temp`), cache-ul navigatoarelor (Chrome, Edge), log-uri și crash dump-uri.
- **Detecție Residual Leftovers**: Scanează și elimină fișierele orfane lăsate în urmă de aplicațiile dezinstalate.

### ⚡ 2. Multi-Method RAM Optimizer & Animated UI
- **Animație Heartbeat & Numărătoare Lină**: Indicatoare de utilizare RAM echipate cu animații interactive de puls (*heartbeat*) și numărătoare lină (`useAnimatedNumber`).
- **Eliberare Memorie Win32**: Curățare avansată prin curățarea *Working Sets* și golirea listei *Standby*.

### 💽 3. Vizualizare Stocare & Discuri (C:, D:, E:)
- **Monitorizare Discuri în Timp Real**: Afișează toate partițiile sistemului cu bare de progres animate și culori dinamice:
  - 🟢 **Verde (<70%)**: Spațiu liber suficient.
  - 🟡 **Galben (70% - 88%)**: Nivel mediu.
  - 🔴 **Roșu Deschis (>88%)**: Stocare aproape plină.

### 🎮 4. Game Booster & Autostart Security Manager
- **Profiluri de Tunat Game Booster**: Presets dedicate pentru gameri (*eSports Competitive*, *Balanced Gamer*, *Streamer Mode*).
- **Manager Startup & Windows Security**: Evaluarea impactului aplicațiilor care pornesc odată cu Windows-ul și verificarea stării de securitate (Defender / Firewall).

### 🌍 5. 100% Tradus în 10 Limbi
- Interfață complet dinamică în **Română, Engleză, Germană, Franceză, Spaniolă, Chineză Simplificată/Tradițională, Japoneză, Turcă și Coreeană**.

## ⚙️ Instrucțiuni de Instalare / Installation Guide

### Varianta 1: Installer `.exe` (Standard)
1. Descarcă fișierul [`Nitrafe One_1.0.0_x64-setup.exe`](./Nitrafe%20One_1.0.0_x64-setup.exe?raw=true).
2. Dă dublu-click pe fișier pentru a porni asistentul de instalare NSIS.
3. Urmează pașii din wizard și lansează **Nitrafe One**.

### Varianta 2: Pachet `.msi` (Instalare Silențioasă / Enterprise)
Pentru instalare rapidă sau silențioasă via PowerShell / Command Prompt:
```powershell
msiexec /i "Nitrafe One_1.0.0_x64_en-US.msi" /qb
```
